# CAP-CAT-260320 :::: https://didactor91.github.io/CAP-CAT-260320/

Web sencilla para comparar rápidamente los resultados del examen del **CAP INICIAL** del **20/03/2026**.

### Opciones disponibles
* **Modalidades:** Mercancías / Viajeros.
* **Modelos:** Modelo A / Modelo B.

### Instrucciones de entrada
* Introducir las respuestas **sin espacios**.
* Marcar como **X** las respuestas en blanco.
* Introducir **100 respuestas** (las 3 preguntas de reserva no se contabilizan).

### Resultados
El sistema determina el estado de **APTO** o **NO APTO** (mínimo 50 puntos). Se muestra la puntuación final y un resumen de las preguntas falladas o en blanco para facilitar la revisión con los enunciados.

A simple web tool to quickly compare the results of the **CAP INITIAL** exam held on **20/03/2026**.

### Available Options
* **Categories:** Goods / Passengers.
* **Versions:** Model A / Model B.

### Input Instructions
* Enter answers **without spaces**.
* Use **X** for blank answers.
* Input **100 answers** (the 3 extra/reserve questions are not included).

### Results
The system determines a **APTO** (pass) or **NO APTO** (fail) status (minimum 50 points). It displays the final score and a summary of missed or blank questions to simplify cross-referencing with the exam paper.
